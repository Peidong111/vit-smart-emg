\# VIT SMART EMG



This project implements a simple EMG-based muscle fatigue monitoring system.



\## Structure



\- `data/raw/`: raw EMG data (not uploaded)

\- `data/processed/`: metadata and processed files

\- `scripts/`: training and inference code

\- `models/`: trained models

\- `outputs/`: prediction results



\## Goal



Build an end-to-end pipeline:



EMG signal -> feature extraction -> model -> fatigue prediction
